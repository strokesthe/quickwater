


# React Waterlogging App

A quick way to log your daily water intake based on your water needs


## How to install

- click on "Code"
- click on "Download Zip"
- drag folder into vs code
- open new terminal
- type "npm install"
- type "npm run dev"
- copy the "Local:" address
- paste the URL into your browser and press enter
- enjoy =)

## Features
- calculates your water needs for the day
- log your water intake
- see your water intake average for the week
- fun UI!


## Tech Stack
- React
- JavaScript
- CSS (Flexbox and Animations)
